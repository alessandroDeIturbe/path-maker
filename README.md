> This is a command for making files and directories faster

### 🏠 [Homepage](https://github.com/alessandroDeIturbe/path-maker)

## Install

```sh
python3 install.py
```

## Usage

```sh
pfm folder/list/to/file [OPTIONAL]root/folder
```

## Run tests

```sh
pfm
```
